# CLI Audio Downsampler

A command-line tool for downsampling audio files with support for multiple formats using FFmpeg.

## Features

- Supports multiple audio formats: MP3, WAV, FLAC, AAC, OGG, M4A, and more
- Converts to 16-bit PCM WAV output
- Simple command-line interface
- Cross-platform (Linux, macOS)

## Installation

### Automatic Installation

chmod +x install.sh
./install.sh
